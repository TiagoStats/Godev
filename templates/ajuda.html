Como Utilizar este Sistema


Menu:


Clicanco aqui você tem acesso ao menu, nele você encontra diversos botões, cada um deles representa uma funcionalidade desses sistema. Estes botões são:



Incluir nova pessoa:


Clicanco neste botão você tem acesso à página de cadastro de uma nova pessoa no sistema. Após clicar neste botão você sera redirecionado para uma página onde deverá inserir o nome e o sobrenome do participante e finalizar seu cadastro clicando no botão "Incluir" que se encontrará logo abaixo do formulário. ATENÇÃO: somente é possível fazer a listagem de salas após ja ter cadastrado ao menos duas pessoas.



Incluir nova sala:


Clicanco neste botão você tem acesso à página de cadastro de uma nova sala no sistema. Após clicar neste botão você sera redirecionado para uma página onde deverá inserir o nome da sala que você deseja cadastrar e informar a capacidade total daquela sala de receber pessoas, então você pode finalizar o cadastro clicando no botão "Incluir" que se encontrará logo abaixo do formulário. ATENÇÃO: somente é possível fazer a listagem de salas após ja ter cadastrado ao menos duas salas.



Incluir nova cafeteria:


Clicanco neste botão você tem acesso à página de cadastro de uma nova cafeteria ou sala de lanche no sistema. Após clicar neste botão você sera redirecionado para uma página onde deverá inserir o nome da sala em questão e finalizar seu cadastro clicando no botão "Incluir" que se encontrará logo abaixo do formulário.



Listar Salas:


Clicanco neste botão você tem acesso à página de listagem de todas as salas cadastrada. Nesta página é possível ver que pessoa está cadastrada em cada sala em cada período do evento e em que cafeteria cada participante está cadastrado para fazer seu intervalo.



Como utilizar este sistema


Clicanco neste botão você tem acesso a esta página que você está agora mesmo, você pode acessá-la a qualquer momento se sentir necessidade de tirar dúvidas.
